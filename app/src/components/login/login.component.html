Login

This is a sample login page. It contains a basic form with a directive for input fields. The form-input-text directive is bound to a FormControl directive and provides input validation for displaying error messages.

To log in, enter user and 123 in the respective input fields. Afterwards, you will be redirected to the dashboard view which is a sample area for authorized users. If you try navigating to the dashboard manually, you will be redirected to this page.

The login component makes use of a service for authorization which will check credentials and save an example token in the sessionStorage. This concept can be extended with real HTTP communication and backend authorization later.
